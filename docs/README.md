## Configuration

The following JSON is the default configuration for the module and can be overridden in your project's `composer.json` file.

```json
{
	"extra": {
		"altis": {
			"modules": {
				"seo": {
					"enabled": true
					},
					"redirects": true
				}
			}
		}
	}
}
```
